Flask Blog

Project information

Flag Blog is a full featured blog style web application built using Flask framework, where different user can make different post which can be regular blog post or they can be nay updates. Anyone can register in this application as a new user or sign-in if already a user. We also have a forgot password option which will let user reset password by getting an email. we also have the ability to update profile pictures. One the homepage we can look at other peoples post, and for our post we can CREATE / DELETE / UPDATE a post.
